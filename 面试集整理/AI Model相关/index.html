    一、机器学习基础¶

    1. 机器学习算法分类？¶

    答案：

    类别 算法示例 特点
    监督学习 线性回归、逻辑回归、SVM、决策树、神经网络 有标签数据
    无监督学习 K-Means、DBSCAN、PCA、Autoencoder 无标签
    半监督学习 标签传播、协同训练 部分标签
    强化学习 Q-Learning、DQN、Policy Gradient 环境交互学习
    迁移学习 Fine-tuning、Domain Adaptation 知识迁移

    2. 过拟合和欠拟合？如何解决？¶

    答案：

    过拟合： 模型在训练集表现好，测试集表现差

    欠拟合： 模型在训练集和测试集表现都不好

    问题 解决方案
    过拟合 更多数据、正则化（L1/L2）、Dropout、Batch Norm、早停、模型简化
    欠拟合 增加模型复杂度、增加特征、减少正则化、训练更久 
    # 正则化
from sklearn.linear_model import Ridge, Lasso
ridge = Ridge(alpha=1.0)  # L2
lasso = Lasso(alpha=1.0)  # L1

# Dropout (PyTorch)
import torch.nn as nn
model = nn.Sequential(
    nn.Linear(784, 256),
    nn.Dropout(0.5),  # 50% dropout
    nn.ReLU(),
    nn.Linear(256, 10)
)

# 早停
from sklearn.model_selection import cross_val_score
# 监控验证集loss，连续n个epoch不下降则停止

    3. 偏差与方差分解？¶

    泛化误差 = 偏差² + 方差 + 噪声:

    指标 含义 表现
    偏差 模型预测值与真实值的偏离 欠拟合时高
    方差 模型预测值的波动 过拟合时高
    噪声 数据本身的随机性 不可消除

    偏差-方差权衡：

            Total Error
            │
            │      /\
            │     /  \
            │    /    \        ← 方差主导
            │   /      \____
            │  /   /\      ‾‾‾
            │ /__/    \____
            │/              ‾‾ ← 偏差主导
            └──────────────────→ 模型复杂度
                  低    中    高

    4. 交叉验证是什么？如何实现？¶

    答案：

    K折交叉验证：

    from sklearn.model_selection import KFold, cross_val_score
from sklearn.ensemble import RandomForestClassifier

# 5折交叉验证
kfold = KFold(n_splits=5, shuffle=True, random_state=42)
model = RandomForestClassifier()

scores = cross_val_score(model, X, y, cv=kfold, scoring='accuracy')
print(f"CV Scores: {scores}")
print(f"Mean: {scores.mean():.4f}, Std: {scores.std():.4f}")

# 分层K折（保持类别比例）
from sklearn.model_selection import StratifiedKFold
skf = StratifiedKFold(n_splits=5)

    留一法（LOO）：

    from sklearn.model_selection import LeaveOneOut
loo = LeaveOneOut()
# 数据量大时计算量大

    5. 归一化和标准化区别？¶

    答案：

    方法 公式 适用场景
    Min-Max归一化 x' = (x-min)/(max-min) 有固定范围，如图像[0,1]
    Z-Score标准化 x' = (x-μ)/σ 数据近似正态分布
    Robust标准化 x' = (x-median)/IQR 有异常值 
    from sklearn.preprocessing import MinMaxScaler, StandardScaler, RobustScaler

# Min-Max归一化
scaler = MinMaxScaler()
X_scaled = scaler.fit_transform(X)

# Z-Score标准化
scaler = StandardScaler()
X_scaled = scaler.fit_transform(X)

# Robust标准化
scaler = RobustScaler()
X_scaled = scaler.fit_transform(X)

    神经网络： 通常用标准化（Z-Score） 距离敏感算法： 必须标准化（SVM、KNN、K-Means） 决策树： 不需要归一化（不依赖梯度）

    二、深度学习¶

    6. 激活函数有哪些？区别？¶

    答案：

    函数 公式 特点 适用场景
    Sigmoid 1/(1+e⁻ˣ) 输出[0,1]，梯度易消失 二分类输出
    Tanh (eˣ-e⁻ˣ)/(eˣ+e⁻ˣ) 输出[-1,1]，梯度易消失 RNN
    ReLU max(0,x) 计算快，梯度不消失，死神经元 CNN/DNN默认
    Leaky ReLU max(0.01x,x) 避免死神经元 替代ReLU
    ELU x if x>0 else α(eˣ-1) 平滑，接近零均值 深层网络
    Softmax eˣᵢ/Σeˣʲ 多分类输出 多分类 
    import torch.nn as nn

nn.ReLU()
nn.LeakyReLU(0.1)
nn.Sigmoid()
nn.Tanh()
nn.Softmax(dim=1)

    梯度消失问题： sigmoid、tanh在两端梯度接近0 ReLU死神经元： 负输入永远0，可使用Leaky ReLU/PReLU

    7. 损失函数有哪些？¶

    答案：

    任务 损失函数 公式/特点
    回归 MSE (y-ŷ)²，均方误差
    回归 MAE |y-ŷ|，绝对误差，对异常值鲁棒
    二分类 Binary Cross-Entropy -[y·log(ŷ) + (1-y)·log(1-ŷ)]
    多分类 Cross-Entropy -Σyᵢ·log(ŷᵢ)
    多分类 Focal Loss -α(1-ŷ)ᵞ·log(ŷ)，解决类别不平衡 
    import torch.nn as nn

nn.MSELoss()              # 回归
nn.L1Loss()               # MAE
nn.BCELoss()              # 二分类
nn.CrossEntropyLoss()     # 多分类（内置softmax）
nn.BCEWithLogitsLoss()    # Sigmoid + BCE
nn.FocalLoss()            # 平衡分类

    8. 优化器有哪些？区别？¶

    答案：

    优化器 特点 超参数
    SGD 随机梯度下降 lr, momentum
    Momentum 动量加速 lr, momentum=0.9
    AdaGrad 自适应学习率（稀疏特征） lr
    RMSProp AdaGrad改进 lr, decay
    Adam Adam=Momentum+RMSProp lr, betas
    AdamW Adam+L2正则 lr, weight_decay 
    import torch.optim as optim

# SGD with Momentum
optimizer = optim.SGD(model.parameters(), lr=0.01, momentum=0.9)

# Adam (默认推荐)
optimizer = optim.Adam(model.parameters(), lr=0.001)

# AdamW (带权重衰减)
optimizer = optim.AdamW(model.parameters(), lr=0.001, weight_decay=0.01)

# 学习率调度
scheduler = optim.lr_scheduler.StepLR(optimizer, step_size=10, gamma=0.1)
scheduler = optim.lr_scheduler.CosineAnnealingLR(optimizer, T_max=50)
scheduler = optim.lr_scheduler.OneCycleLR(optimizer, max_lr=0.01)

    推荐： 首选AdamW，NLP任务常用

    9. Batch Normalization作用？¶

    答案：

    作用： 对每一层的输入进行标准化，加速训练

    公式：

    μ = mean(x)
σ² = var(x)
x_norm = (x - μ) / √(σ² + ε)
y = γ * x_norm + β  # 可学习的缩放和平移

    好处：

    • 加速收敛
    • 允许更高学习率
    • 减少对初始化的依赖
    • 正则化效果
    import torch.nn as nn

nn.BatchNorm1d(256)  # 用于Linear输出
nn.BatchNorm2d(64)   # 用于Conv2d输出

    注意事项：

    • 训练时用batch统计，推理时用移动平均
    • Batch小可能导致统计不稳定
    • 图像任务BatchNorm，NLP任务常用LayerNorm

    10. Dropout作用？如何实现？¶

    答案：

    作用： 训练时随机丢弃神经元，减少过拟合

    原理： 相当于训练多个模型做集成

    import torch.nn as nn

nn.Dropout(p=0.5)  # 50%丢弃率

    注意事项：

    • 训练时启用，推理时禁用
    • 常用丢弃率：0.2-0.5
    • 丢弃率太小没效果，太大欠学习

    变体：

    方法 特点
    Dropout 随机丢弃
    Dropout2d 丢弃整个通道
    Spatial Dropout Dropout2d的特例
    DropPath 丢弃整个路径（ViT）
    Stochastic Depth 随机丢弃残差块

    11. CNN常用组件？¶

    答案：

    组件 作用
    卷积层 提取特征
    池化层 下采样，减少计算，增加感受野
    全连接层 分类/回归
    1x1卷积 通道数变换
    Depthwise卷积 逐通道卷积，减少计算量
    空洞卷积 增大感受野 
    import torch.nn as nn

nn.Conv2d(in_channels=3, out_channels=64, kernel_size=3, stride=1, padding=1)
nn.MaxPool2d(kernel_size=2, stride=2)
nn.AvgPool2d(kernel_size=2)
nn.AdaptiveAvgPool2d((1, 1))  # 自适应池化到固定尺寸

    经典架构：

    模型 特点
    LeNet 最早的CNN
    AlexNet 深度CNN，ReLU、GPU
    VGG 小卷积核(3x3)堆叠
    GoogLeNet Inception模块
    ResNet 残差连接
    DenseNet 密集连接
    EfficientNet 复合缩放

    12. RNN/LSTM/GRU区别？¶

    答案：

    RNN问题： 梯度消失/爆炸，无法捕获长距离依赖

    LSTM（Long Short-Term Memory）：

    门控机制：
- 遗忘门：决定丢弃什么信息
- 输入门：决定更新什么信息
- 输出门：决定输出什么信息

公式：
f = σ(Wf·[h_{t-1}, x_t])
i = σ(Wi·[h_{t-1}, x_t])
C~_t = tanh(Wc·[h_{t-1}, x_t])
C_t = f * C_{t-1} + i * C~_t
o = σ(Wo·[h_{t-1}, x_t])
h_t = o * tanh(C_t)

    GRU（Gated Recurrent Unit）：

    更新门：合并了输入门和遗忘门
z = σ(Wz·[h_{t-1}, x_t])
r = σ(Wr·[h_{t-1}, x_t])
h~_t = tanh(W·[r * h_{t-1}, x_t])
h_t = (1-z) * h_{t-1} + z * h~_t

    对比：

    特性 RNN LSTM GRU
    参数量 少 多 中
    长依赖 差 好 好
    计算复杂度 低 高 中
    效果 差 好 相当

    Transformer替代： 现在更多用Transformer处理序列

    13. 梯度消失和梯度爆炸？¶

    原因：

    • 链式法则导致梯度连乘
    • 激活函数在两端梯度接近0（梯度消失）
    • 权重初始化过大（梯度爆炸）

    解决方案：

    方法 说明
    残差连接 ResNet
    门控机制 LSTM、GRU
    合适激活函数 ReLU
    合适初始化 He初始化
    梯度裁剪 torch.nn.utils.clip_grad_norm_
    BatchNorm 稳定梯度 
    # 梯度裁剪
torch.nn.utils.clip_grad_norm_(model.parameters(), max_norm=1.0)

# He初始化
nn.Linear(in, out).apply(init.kaiming_normal_)

    三、Transformer与BERT¶

    14. Transformer架构？¶

    答案：

    核心组件：

    Input → Embedding → Positional Encoding →
    Encoder Layer × N → Encoder Output
                                    ↓
Input → Embedding → Positional Encoding →
    Decoder Layer × N → Linear → Softmax → Output

    Encoder Layer：

    • Multi-Head Self-Attention
    • Add & Norm
    • Feed Forward
    • Add & Norm

    Decoder Layer：

    • Masked Multi-Head Self-Attention
    • Add & Norm
    • Encoder-Decoder Attention
    • Add & Norm
    • Feed Forward
    • Add & Norm

    注意力机制：

    Attention(Q, K, V) = softmax(QK^T / √d_k) V

    # PyTorch实现
import torch.nn as nn

class TransformerEncoderLayer(nn.Module):
    def __init__(self, d_model, nhead, dim_feedforward=2048):
        super().__init__()
        self.self_attn = nn.MultiheadAttention(d_model, nhead)
        self.ff = nn.Sequential(
            nn.Linear(d_model, dim_feedforward),
            nn.ReLU(),
            nn.Linear(dim_feedforward, d_model)
        )
        self.norm1 = nn.LayerNorm(d_model)
        self.norm2 = nn.LayerNorm(d_model)

    def forward(self, x):
        x = self.norm1(x + self.self_attn(x, x, x)[0])
        x = self.norm2(x + self.ff(x))
        return x

    答案：

    方法 全参数量 显存需求 效果 适用场景
    Full Fine-tune 全部 高 最好 有足够资源
    LoRA 低秩矩阵 中 接近全参 推荐首选

    Prefix Tuning 前缀 中 较好 多任务
    Prompt Tuning soft prompt 低 一般 多任务

    LoRA原理：

    原权重: W₀ ∈ R^(d×k)
更新: ΔW = BA, B∈R^(d×r), A∈R^(r×k), r << min(d,k)
新权重: W = W₀ + ΔW
训练时只更新A和B

# Perplexity
import math
def perplexity(text, model):
    n = len(text)
    loss = 0
    for i in range(n):
        probs = model.predict(text[:i+1])
        loss -= math.log(probs[text[i]])
    return math.exp(loss / n)

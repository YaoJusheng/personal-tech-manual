    1.Mac基础配置¶

    1.0.brew¶

    安装参考: brew安装

    # Homebrew是 mac的包管理器
/usr/bin/ruby -e "$(curl -fsSL https://cdn.jsdelivr.net/gh/ineo6/homebrew-install/install)"
# 替换 brew.git
git -C "$(brew --repo)" remote set-url origin https://mirrors.ustc.edu.cn/brew.git
# 替换 homebrew-core.git
git -C "$(brew --repo homebrew/core)" remote set-url origin https://mirrors.ustc.edu.cn/homebrew-core.git

# bash 下
echo 'export HOMEBREW_BOTTLE_DOMAIN=https://mirrors.ustc.edu.cn/homebrew-bottles' >> ~/.bash_profile
source ~/.bash_profile
# zsh 下
echo 'export HOMEBREW_BOTTLE_DOMAIN=https://mirrors.ustc.edu.cn/homebrew-bottles' >> ~/.zshrc
source ~/.zshrc

# 更新
brew update

    1.1.包管理¶

    # [1]--- python ---
# 安装pyenv
brew install pyenv
# 更新
brew update && brew upgrade pyenv

# [2]--- ruby ---
# 参考: https://www.jianshu.com/p/e36c0a1c6b49#

# 2.安装mpapis公钥

# 3.安装最新版本的Ruby的RVM
curl -sSL https://get.rvm.io | bash -s stable --ruby

# [3]--- node ---
brew install node
npm install -g n
# or
brew install n
# 安装最新
sudo n latest/stable

    1.5.Mac地址¶

    1.5.1.查找网络接口名称¶

    $ networksetup -listallhardwareports
# 断开网络连接
$ networksetup -setairportpower en0 off
# 重新启用网络接口
$ networksetup -setairportpower en0 on

    1.5.2.查看网络配置¶

    ifconfig en0
ifconfig en0 | grep ether

    1.5.3.修改MAC地址¶

    • 在修改MAC地址之前，可以使用airport命令断开WiFi连接
    sudo /System/Library/PrivateFrameworks/Apple80211.framework/Versions/Current/Resources/airport -z

    3.6.2.leap.nvim¶

    • 推荐度：★★★★★

    • 简介：leap.nvim可以让你的光标快速在可视范围内跳转。只需要按下引导键（自定义的第一个键），然后输入你想跳转到的地方的相邻2个字符，如有重复，输入最后一个提示字符即可。可以配合其它neovim快捷键一起使用，如"c", "d", "y"

        vim.keymap.set({ "x", "o", "n" }, "W", "<Plug>(leap-from-window)")
    end,
},


-- 快捷键介绍：
-- r：在当前光标往右往下查找。
-- R：在当前光标往左往上查找。
-- W：多窗口情况下，可快速跳转到其它窗口的内容。

# 7.忽略大小写(-i)

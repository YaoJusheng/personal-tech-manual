    一、文字符号¶

    在日志输出中使用符号能让信息更直观、易读。下面以 Markdown 格式详细列举一些常见符号及其含义，同时给出使用示例代码。

    1.✅ 成功相关¶

    符号 Unicode 编码 说明 示例场景
    ✅ U+2705 表示操作成功完成，常用于确认某个任务、流程顺利结束。 数据库连接成功、文件上传完成等。需求已上线 ✅
    🎉 U+1F389 庆祝成功，适用于重要功能上线、重大任务完成等值得庆祝的场景。 新版本发布、系统完成关键升级。Sprint 100% 完成 🎉
    🌟 U+1F31F 突出显示成功的亮点，可用于标记表现优秀的操作或成果。 某接口响应时间创历史新低、某模块性能大幅提升。
    🏆 U+1F3C6 冠军/最佳 质量榜首 🏆
    🚀 U+1F680 发布成功 版本 v2.0.0 已发布 🚀
    ✨ U+2728 闪亮新功能 新特性上线 ✨
    🎯 U+1F3AF 目标达成 OKR 全部命中 🎯
    🥳 U+1F973 团队狂欢 项目庆功 🥳
    🏁 U+1F3C1 冲刺终点 迭代结束 🏁

    2.❌ 错误相关¶

    符号 Unicode 编码 说明 示例场景
    ❌ U+274C 表示操作失败或出现错误，是错误提示中常用的符号。 数据库查询失败、文件读写异常。
    🚫 U+1F6AB 禁止、阻止的意思，可用于表示某个操作被禁止或受到限制。 用户无权限访问某资源、系统达到最大连接数限制。
    ⚠️ U+26A0 FE0F 警告符号，提示存在潜在的问题或风险，但尚未导致严重错误。 磁盘空间不足、内存使用率过高。
    🔥 U+1F525 表示系统出现严重故障，类似“着火”般紧急，需要立即处理。 服务崩溃、数据库宕机。
    🔴 U+1F534 红灯/致命 线上故障 🔴
    ❗ U+2757 紧急 严重缺陷 ❗
    🐛 U+1F41B Bug 发现新 Bug 🐛
    💥 U+1F4A5 爆炸/崩溃 服务宕机 💥

    3.ℹ️ 信息相关¶

    符号 Unicode 编码 说明 示例场景
    ℹ️ U+2139 FE0F 提供信息，用于展示一些重要的提示、说明或状态信息。 系统当前版本号、配置参数信息。
    📢 U+1F4E3 广播、通知，可用于发布系统公告、版本更新等消息。 系统维护通知、新版本特性介绍。
    📋 U+1F4CB 表示记录、清单，适用于输出日志记录、配置信息等。 打印系统操作日志、显示当前用户列表。
    📣 U+1F4E3 重要通知 维护通知 📣
    📰 U+1F4F0 新闻/更新 产品更新 📰
    📄 U+1F4C4 文档/文本信息 API 文档 📄
    📑 U+1F4D1 标签/目录 功能目录 📑
    🗂️ U+1F5C2 索引/归档 资料归档 🗂️
    📊 U+1F4CA 数据信息 统计图表 📊
    📈 U+1F4C8 趋势信息 增长趋势 📈
    📉 U+1F4C9 下降趋势 风险预警 📉
    📅 U+1F4C5 时间信息 发布日期 📅
    🕒 U+1F552 时间点 定时提醒 🕒
    📝 U+1F4DD 记录/备注 会议纪要 📝
    📥 U+1F4E5 接收信息 需求收集 📥
    📤 U+1F4E4 发送信息 结果输出 📤

    📝 U+1F4DD 待记录或待撰写 会议纪要 📝

    🚧 U+1F6A7 阻塞 / 施工中 环境搭建 🚧
    🎯 U+1F3AF 目标 / 里程碑 本迭代里程碑 🎯
    ⚠️ U+26A0 警告 / 风险 依赖服务异常 ⚠️
    ❗ U+2757 紧急 线上故障 ❗
    🚀 U+1F680 上线 / 发布 版本 v2.3.0 已发布 🚀
    🧪 U+1F9EA 测试阶段 自动化回归 🧪
    🔍 U+1F50D 评审 / 复盘 代码评审 🔍
    📈 U+1F4C8 进度增长 本周完成率 +20% 📈
    📊 U+1F4CA 数据看板 Sprint 燃尽图 📊
    🎉 U+1F389 庆祝 / 成功 Sprint 达成 100% 🎉
    📥 U+1F4E5 需求流入 新需求 📥
    📤 U+1F4E4 需求交付 功能已交付 📤
    🏷️ U+1F3F7 标签 / 分类 高优先级 🏷️
    🗂️ U+1F5C2 归档 / 整理 文档已归档 🗂️

    5.⏳进度相关¶

    符号 Unicode 编码 说明 示例场景
    🔄 U+1F504 循环/迭代 需求变更，进入下一轮迭代 🔄
    ⏳ U+23F3 计时进行中, 表示操作正在进行中，处于等待或处理状态。 任务开发中 ⏳, 文件下载中、数据正在导入。
    ⏸️ U+23F8 暂停 等待评审，任务暂停 ⏸️
    ⏯️ U+23EF 继续/恢复 评审完成，任务继续 ⏯️
    ⏹️ U+23F9 终止 需求取消，任务终止 ⏹️
    📈 U+1F4C8 上升/进度增长 本周完成率 +15% 📈
    📉 U+1F4C9 下降/进度倒退 缺陷激增，完成率 -8% 📉
    🚧 U+1F6A7 阻塞/施工中 环境部署 🚧
    🛤️ U+1F6E4 路线图/里程碑 发布计划 🛤️
    🛠️ U+1F6E0 开发/构建 功能开发 🛠️
    🔧 U+1F527 修复/调优 修复缺陷 🔧
    ⚙️ U+2699 配置/集成 流水线配置 ⚙️
    🧪 U+1F9EA 测试阶段 回归测试 🧪
    🚀 U+1F680 上线/发布, 象征启动、开始，可用于标记某个进程、服务的启动。 版本已发布 🚀, 服务器启动、定时任务开始执行。
    🎯 U+1F3AF 目标达成 迭代目标完成 🎯
    🏁 U+1F3C1 冲刺终点 Sprint 结束 🏁
    📊 U+1F4CA 看板/燃尽图 进度看板 📊
    📅 U+1F4C5 截止日期 上线日 📅 2024-12-01
    🗓️ U+1F5D3 日程/时间线 项目日程 🗓️
    🎉 U+1F389 庆祝完成 100% 达成率 🎉
    🛑 U+1F6D1 停止、中断，用于表示某个操作被手动或自动停止。 手动停止服务、任务因超时中断。

    6.🔐 安全相关¶

    符号 Unicode 编码 说明 示例场景
    🔒 U+1F512 表示安全、加密，可用于提示某个操作或数据处于安全保护状态。 敏感数据加密存储、用户登录会话已加密。
    🔓 U+1F513 解锁，用于表示安全限制被解除，操作可以继续进行。 用户输入正确密码解锁账户、安全模式解除。
    👮 U+1F46E 代表安全防护，可用于记录安全检查、权限验证等操作。 系统进行安全扫描、用户权限验证通过。
    🔐 U+1F510 带钥匙的锁 密钥存储 🔐
    🛡️ U+1F6E1 盾牌 防御攻击 🛡️
    🔑 U+1F511 密钥 API 密钥 🔑
    🚫 U+1F6AB 禁止/拒绝 未授权访问 🚫
    ⚠️ U+26A0 安全警告 漏洞警报 ⚠️
    🚨 U+1F6A8 紧急安全事件 入侵检测 🚨

    7.🌐 网络相关¶

    符号 Unicode 编码 说明 示例场景

    📶 U+1F4F6 代表网络信号强度，可用于显示网络连接质量。 显示 Wi-Fi 信号强度、蜂窝网络信号状态。
    📡 U+1F4E1 表示与网络信号传输、接收有关，可用于网络通信相关操作提示。 正在接收网络数据、发送网络请求。
    🔗 U+1F517 链接/连接 建立连接 🔗
    🔌 U+1F50C 插拔连接 断开重连 🔌
    🛰️ U+1F6F0 卫星/远程 远程网关 🛰️
    🕸️ U+1F578 网络拓扑 服务网格 🕸️
    ⚡ U+26A1 网络加速 高并发 ⚡

    8.💾 数据相关¶

    符号 Unicode 编码 说明 示例场景
    💾 U+1F4BE 表示数据存储操作，如保存、写入数据到存储设备。 数据成功保存到硬盘、缓存数据写入。
    📤 U+1F4E4 表示数据上传操作，将本地数据发送到远程服务器。 文件上传到云存储、日志数据上传。
    📥 U+1F4E5 表示数据下载操作，从远程服务器获取数据到本地。 下载软件更新包、获取数据库备份文件。
    🗃️ U+1F5C3 表示数据存储容器，可用于提示数据在存储中的位置或操作存储中的数据。 数据归档到存储库、从数据库表中读取数据。
    📊 U+1F4CA 数据图表 实时报表 📊
    📈 U+1F4C8 数据增长 日志量激增 📈
    📉 U+1F4C9 数据下降 存储回收 📉
    🗂️ U+1F5C2 数据归档 冷存归档 🗂️
    🔍 U+1F50D 数据检索 全文搜索 🔍
    🧬 U+1F9EC 数据链 区块链 🧬
    🔢 U+1F522 数值数据 指标统计 🔢

    9.🕒时间相关¶

    符号 Unicode 编码 说明 示例场景
    ⏰ U+23F0 表示定时任务、闹钟提醒，可用于标记定时执行的操作。 定时备份任务开始执行、定时清理缓存。

    package main

import (
    "log"
    "time"
)

func printEmoji() {
    // 启动服务
    log.Print("🚀 Starting the application...")
    log.Print("🌐 Network service is initializing...")

    // 模拟成功操作
    log.Print("✅ Database connection established successfully.")
    log.Print("💾 User data saved to disk.")

    // 模拟重要功能上线
    log.Print("🎉 New feature 'Real - time Chat' is now live!")

    // 模拟性能亮点
    log.Print("🌟 API response time improved by 50%.")

    // 模拟网络相关
    log.Print("📶 Wi - Fi signal strength is excellent.")
    log.Print("📡 Sending authentication request to server...")

    // 模拟警告信息
    log.Print("⚠️  Disk space is running low, only 10% remaining.")
    log.Print("🧠 CPU usage is over 90%, potential performance issue.")

    // 模拟操作失败
    log.Print("❌ File upload failed due to network issues.")

    // 模拟禁止操作
    log.Print("🚫 Access denied: insufficient permissions.")

    // 模拟严重故障
    log.Print("🔥 Database service crashed unexpectedly.")

    // 提供系统信息
    log.Print("ℹ️  Current system version: 1.2.3")

    // 发布系统通知
    log.Print("📢 System maintenance will be performed at 2 AM.")

    // 记录操作日志
    log.Print("📋 User 'admin' logged in at 2025-07-03 21:30:00.")

    // 显示操作进度
    log.Print("⏳ File download in progress: 75% completed.")

    // 时间相关
    start := time.Now()
    // 模拟一些操作
    time.Sleep(2 * time.Second)
    elapsed := time.Since(start)
    log.Print("⏱️  Operation took ", elapsed, " to complete.")
    log.Print("⏰ Daily backup task started.")

    // 交互相关
    log.Print("👋 Welcome, user 'admin'.")
    log.Print("💬 New message from support: 'System update available'.")

    // 停止服务
    log.Print("🛑 Application stopped by user command.")

    // 安全提示
    log.Print("🔒 User session data is encrypted.")
    log.Print("🔓 Account unlocked after password verification.")
    log.Print("👮 Security check completed, no threats detected.")

    // 资源相关
    log.Print("⚡ Switching to high - performance mode.")
    log.Print("💧 Network traffic consumption is high this hour.")
}

func printU2Emoji() {
    // 4 字节 UTF-8 转义
    fmt.Println("倒计时 \U000023F3") // ⏳
    // 2 字节 UTF-8 转义
    fmt.Println("火箭 \u26F5")      // ⛵
}

func main() {
    printU2Emoji()
    printEmoji()
}

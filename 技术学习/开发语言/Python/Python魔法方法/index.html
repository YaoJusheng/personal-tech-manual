    1. 对象生命周期 (Lifecycle)¶

    方法 签名 描述与使用场景
    __new__ __new__(cls, *args, **kwargs) 构造器。在 __init__ 之前调用，负责创建并返回实例。是静态方法。
    场景：单例模式；继承不可变类型（如 tuple, str）；元类编程。
    __init__ __init__(self, *args, **kwargs) 初始化器。在实例创建后调用，负责初始化属性。
    场景：常规类的属性赋值。
    __del__ __del__(self) 析构器。在对象引用计数归零被垃圾回收时调用。
    场景：极少主动使用。警告：不可靠，依赖 GC 机制；资源释放应优先使用上下文管理器。

    2. 字符串与字节表示 (Representation)¶

    方法 签名 描述与使用场景
    __repr__ __repr__(self) 官方字符串表示。目标是“无歧义”，尽量返回合法的 Python 代码字符串。
    场景：调试 (repr())，交互式命令行输出。
    __str__ __str__(self) 用户字符串表示。目标是“可读性”。
    场景：print(), str(), 格式化字符串 %s。
    __format__ __format__(self, format_spec) 格式化协议。自定义格式化字符串的行为。
    场景：f"{obj:fmt}" 或 format(obj, "fmt")。
    __bytes__ __bytes__(self) 字节表示。
    场景：bytes(obj)。
    __fspath__ __fspath__(self) 文件系统路径表示。
    场景：使对象可被 os.path 或 open() 识别为路径（如 pathlib.Path）。

    3. 比较与散列 (Comparison & Hashing)¶

    方法 签名 描述
    __eq__ __eq__(self, other) == (Equal)。默认比较 id()。
    __ne__ __ne__(self, other) != (Not Equal)。
    __lt__ __lt__(self, other) < (Less Than)。此方法也是 sort() 的基础。
    __gt__ __gt__(self, other) > (Greater Than)。
    __le__ __le__(self, other) <= (Less or Equal)。
    __ge__ __ge__(self, other) >= (Greater or Equal)。
    __hash__ __hash__(self) 散列值。
    场景：使对象可用作字典键或集合元素。仅当对象不可变且实现了 __eq__ 时实现。
    __bool__ __bool__(self) 布尔值。
    场景：if obj: 或 bool(obj)。默认依据 __len__ 或为 True。

    4. 属性访问与描述符 (Attribute Access & Descriptors)¶

    属性控制¶

    方法 签名 描述与使用场景
    __getattr__ __getattr__(self, name) 仅当属性查找失败时调用。
    场景：动态代理、延迟加载、REST API 包装器。
    __getattribute__ __getattribute__(self, name) 无条件拦截所有属性访问。
    场景：极少使用，易引起无限递归。需通过 super() 调用。
    __setattr__ __setattr__(self, name, value) 拦截属性赋值 (obj.x = v)。
    场景：类型检查、只读属性实现。
    __delattr__ __delattr__(self, name) 拦截属性删除 (del obj.x)。
    __dir__ __dir__(self) 自定义 dir(obj) 的返回值。
    场景：为动态属性提供自动补全支持。

    描述符协议 (Descriptors)¶

    用于托管属性访问（如 @property 也是基于此实现）。

    方法 签名 描述
    __get__ __get__(self, instance, owner) 获取属性值。
    __set__ __set__(self, instance, value) 设置属性值。
    __delete__ __delete__(self, instance) 删除属性值。
    __set_name__ __set_name__(self, owner, name) (Python 3.6+) 类创建时调用，告知描述符其在类中的接收变量名。
    场景：ORM 字段名自动绑定。

    5. 容器与序列 (Containers & Sequences)¶

    方法 签名 描述与使用场景
    __len__ __len__(self) len(obj)。
    __length_hint__ __length_hint__(self) operator.length_hint(obj)。提供长度的估算值，用于优化内存分配（如 list(iter)）。
    __getitem__ __getitem__(self, key) obj[key]。实现索引访问、切片或键查询。
    __setitem__ __setitem__(self, key, value) obj[key] = value。
    __delitem__ __delitem__(self, key) del obj[key]。
    __contains__ __contains__(self, item) item in obj。若未实现，会回退迭代检查。
    __missing__ __missing__(self, key) 仅用于 dict 子类。当键不存在时调用。
    场景：collections.defaultdict。
    __reversed__ __reversed__(self) reversed(obj)。返回反向迭代器。

    6. 迭代器 (Iterators)¶

    方法 签名 描述与使用场景
    __iter__ __iter__(self) 返回迭代器对象自身。for x in obj 时调用。
    __next__ __next__(self) (Python 3) 返回下一个元素。无元素时抛出 StopIteration。
    场景：自定义迭代逻辑。

    7. 数学运算 (Numeric Operators)¶

    Python 提供了三类运算符重载：

    1. 常规 (Normal): a + b -> a.__add__(b)
    2. 反向 (Reflected/Right): a + b，若 a 不支持，调 b.__radd__(a)
    3. 原地 (In-place): a += b -> a.__iadd__(b)
    运算 常规方法 反向方法 (r) 原地方法 (i)
    + __add__ __radd__ __iadd__
    - __sub__ __rsub__ __isub__
    * __mul__ __rmul__ __imul__
    / __truediv__ __rtruediv__ __itruediv__
    // __floordiv__ __rfloordiv__ __ifloordiv__
    % __mod__ __rmod__ __imod__
    obj ** p __pow__ __rpow__ __ipow__
    obj @ other __matmul__ __rmatmul__ __imatmul__
    divmod(a, b) __divmod__ __rdivmod__ __idivmod__

    位运算: __and__ (&), __or__ (|), __xor__ (^), __lshift__ (<<), __rshift__ (>>), __invert__ (~).

    一元运算与类型转换:

    • __pos__: +obj
    • __neg__: -obj
    • __abs__: abs(obj)
    • __invert__: ~obj

    类型转换与数学函数:

    • __int__, __float__, __complex__: 类型转换。
    • __index__: 无损转换为整数（用于切片索引）。
    • __round__ (round()), __trunc__ (math.trunc()), __floor__ (math.floor()), __ceil__ (math.ceil()).

    8. 上下文管理 (Context Managers)¶

    用于 with 语句。

    方法 签名 描述
    __enter__ __enter__(self) 进入上下文。返回值绑定到 as 后的变量。
    __exit__ __exit__(self, exc_type, exc_val, exc_tb) 退出上下文。处理异常（返回 True 可吞掉异常）。

    9. 可调用对象 (Callables)¶

    方法 签名 描述
    __call__ __call__(self, *args, **kwargs) 使实例像函数一样被调用 obj()。
    场景：装饰器类、保持状态的回调函数。

    10. 异步编程 (Async IO)¶

    Python 3.5+ 引入。

    方法 签名 描述
    __await__ __await__(self) 使对象可被 await（通常由框架实现）。
    __aiter__ __aiter__(self) 异步迭代器 (async for 启动时调用)。返回一个实现了 __anext__ 的对象。
    __anext__ __anext__(self) 异步获取下一个值。无元素时抛出 StopAsyncIteration。
    __aenter__ __aenter__(self) 异步上下文进入 (async with)。必须返回 awaitable (通常是 async def)。
    __aexit__ __aexit__(self, ...) 异步上下文退出 (async with)。必须返回 awaitable。

    11. 类创建与元编程 (Class Construction)¶

    方法 描述
    __init_subclass__(cls) (Python 3.6+) 当类被继承时在父类中调用。替代简单的元类需求。
    场景：注册子类、验证子类属性。
    __class_getitem__(cls, item) (Python 3.7+) 支持类级别的下标访问 Cls[key]。
    场景：实现泛型类型提示（如 List[int]）。
    __mro_entries__(self, bases) (Python 3.9+) 允许普通类伪装成基类，参与 MRO 计算。极少使用。
    __instancecheck__(cls, instance) 由元类实现。自定义 isinstance(instance, cls) 的判定逻辑。
    __subclasscheck__(cls, subclass) 由元类实现。自定义 issubclass(subclass, cls) 的判定逻辑。
    __prepare__(metacls, name, bases) 元类方法。返回用于类命名空间的字典（如 OrderedDict）。

    12. 其他特殊方法¶

    • __match_args__: (Python 3.10+) 定义位置参数的名称，支持 match case 结构化模式匹配。
    • __copy__ / __deepcopy__: 定义 copy.copy() 和 copy.deepcopy() 的行为。
    • __getstate__ / __setstate__: 为 pickle 序列化定制对象状态。
    • __reduce__ / __reduce_ex__: 为 pickle 提供更底层的序列化协议。
    • __getnewargs__ / __getnewargs_ex__: 为不可变对象在反序列化时提供构造参数。
    • __slots__: 限制实例属性，节省内存。
    • __sizeof__: sys.getsizeof() 调用，返回内存占用字节数。

    13. 特殊用法与场景案例 (Special Use Cases)¶

    本节展示魔术方法的“高阶”用法，涵盖同步与异步场景的对比实现。

    13.1 生成器与迭代器 (__iter__ / __next__)¶

    13.1.1 同步迭代器¶

    如果一个类包含状态并需要生成序列，可以同时实现 __iter__ 和 __next__。

    示例：斐波那契数列生成器:

    class Fibonacci:
    def __init__(self, limit):
        self.limit = limit
        self.a, self.b = 0, 1
        self.count = 0

    def __iter__(self):
        return self  # 迭代器通常返回自身

    def __next__(self):
        if self.count >= self.limit:
            raise StopIteration
        result = self.a
        self.a, self.b = self.b, self.a + self.b
        self.count += 1
        return result

# 使用
for num in Fibonacci(5):
    print(num)  # 0, 1, 1, 2, 3

    13.1.2 异步迭代器 (Async Iterator)¶

    用于需要异步获取数据的场景（如网络请求、数据库查询）。

    示例：异步数据抓取器:

    import asyncio

class AsyncFetcher:
    def __init__(self, count):
        self.count = count
        self.current = 0

    def __aiter__(self):
        return self

    async def __anext__(self):
        if self.current >= self.count:
            raise StopAsyncIteration

        # 模拟异步 IO 操作
        await asyncio.sleep(0.1)
        self.current += 1
        return f"Data chunk {self.current}"

# 使用
async def main():
    async for chunk in AsyncFetcher(3):
        print(chunk)

# asyncio.run(main())

    13.2 资源管理与上下文 (__enter__ / __exit__)¶

    13.2.1 同步上下文管理器¶

    除文件操作外，with 语句常用于计时、锁、特定环境配置。

    示例：执行计时器:

    import time

class Timer:
    def __enter__(self):
        self.start = time.time()
        return self

    def __exit__(self, exc_type, exc_val, exc_tb):
        self.end = time.time()
        print(f"Code block took: {self.end - self.start:.4f}s")
        # 返回 False 表示不压制异常

# 使用
with Timer():
    sum(range(1000000))

    13.2.2 异步上下文管理器 (Async Context Manager)¶

    处理异步资源的连接与关闭。

    示例：模拟异步数据库连接:

    class AsyncDBConnection:
    def __init__(self, db_url):
        self.db_url = db_url

    async def __aenter__(self):
        print(f"Connecting to {self.db_url}...")
        await asyncio.sleep(0.1)  # 模拟连接耗时
        return self

    async def __aexit__(self, exc_type, exc_val, exc_tb):
        print("Closing connection...")
        await asyncio.sleep(0.1)
        print("Closed.")

    async def query(self, sql):
        return f"Result for {sql}"

# 使用
async def main():
    async with AsyncDBConnection("db://localhost") as db:
        result = await db.query("SELECT *")
        print(result)

    13.3 管道链式调用 (__or__ / __ror__)¶

    利用 | 运算符实现函数式编程中的管道流（Pipe）。

    13.3.1. 同步管道¶

    示例：数据处理管道:

    class Pipe:
    def __init__(self, func):
        self.func = func

    def __ror__(self, other):
        # other | self -> self.func(other)
        return self.func(other)

@Pipe
def clean(data):
    return data.strip().lower()

@Pipe
def tokenize(data):
    return data.split()

# 使用
raw_data = "  Hello Python World  "
result = raw_data | clean | tokenize
print(result)  # ['hello', 'python', 'world']

    13.3.2 异步管道 (async pipe)¶

    当每一步都是异步 IO 时，可以让 __ror__ 接受协程并按顺序 await。

    示例：异步清洗 + 异步分词:

    import asyncio

class AsyncPipe:
    def __init__(self, func):
        self.func = func

    def __ror__(self, other):
        async def runner():
            value = other
            if asyncio.iscoroutine(value):
                value = await value
            return await self.func(value)
        return runner()

@AsyncPipe
async def a_clean(data):
    await asyncio.sleep(0.05)
    return data.strip().lower()

@AsyncPipe
async def a_tokenize(data):
    await asyncio.sleep(0.05)
    return data.split()

async def main():
    result = await ("  Hello Async Pipe  " | a_clean | a_tokenize)
    print(result)

# asyncio.run(main())

    13.4 单例模式 (__new__)¶

    利用 __new__ 控制实例创建过程，实现单例。

    示例：线程安全的单例:

    class Singleton:
    _instance = None

    def __new__(cls, *args, **kwargs):
        if cls._instance is None:
            # 实际场景可能需要加锁
            cls._instance = super().__new__(cls)
        return cls._instance

s1 = Singleton()
s2 = Singleton()
print(s1 is s2)  # True

    13.5 路径拼接 DSL (__truediv__)¶

    利用 / 运算符实现直观的路径拼接。

    示例:

    import os

class SmartPath:
    def __init__(self, path):
        self.path = path

    def __truediv__(self, other):
        return SmartPath(os.path.join(self.path, str(other)))

    def __repr__(self):
        return f"SmartPath('{self.path}')"

p = SmartPath("/usr")
print(p / "local" / "bin")  # SmartPath('/usr/local/bin')

    13.6 动态属性代理 (__getattr__)¶

    当访问不存在的属性时触发，常用于 API 封装。

    __getattr__ 是一个特殊的方法，在访问对象中不存在的属性时自动调用。只有当通过 __getattribute__ 无法找到属性时才会调用 __getattr__。

    使用场景:

    1. 动态属性访问
    2. 用于访问存储在对象内部的数据结构中的值
    3. 提供默认值而不是抛出 AttributeError
    4. 代理模式
    5. 将属性访问委托给另一个对象
    6. 配置对象
    7. 创建可以动态访问配置项的对象
    8. API包装器
    9. 用于构建灵活的API客户端

    示例：

    import asyncio
import functools
from typing import Any, Callable


class DynamicAttributeHandler:
    """
    演示 __getattr__ 魔法方法的使用场景
    当访问对象不存在的属性时，会调用此方法
    """

    def __init__(self):
        self.existing_attr = "这是一个存在的属性"
        self.data = {
            'name': 'John',
            'age': 30,
            'city': 'Beijing'
        }

    def __getattr__(self, name: str) -> Any:
        """
        当访问不存在的属性时被调用

        Args:
            name: 属性名

        Returns:
            属性值或默认值
        """
        print(f"访问不存在的属性: {name}")

        # 动态返回数据中的值
        if name in self.data:
            return self.data[name]

        # 提供默认值
        return f"默认值_{name}"

    def __getattribute__(self, name: str) -> Any:
        """
        在每次访问属性时都会被调用（比__getattr__优先级更高）
        """
        # 避免无限递归
        if name.startswith('_') or name in ['data', '__dict__', '__class__']:
            return object.__getattribute__(self, name)

        print(f"通过 __getattribute__ 访问: {name}")
        try:
            # 尝试获取实际属性
            return object.__getattribute__(self, name)
        except AttributeError:
            # 如果属性不存在，则调用 __getattr__
            return self.__getattr__(name)

class DynamicMethodGenerator:
    """
    使用 __getattr__ 动态生成方法的示例
    """

    def __init__(self):
        self.operations = {
            'add': lambda x, y: x + y,
            'sub': lambda x, y: x - y,
            'mul': lambda x, y: x * y,
            'div': lambda x, y: x / y if y != 0 else float('inf')
        }

    def __getattr__(self, name: str) -> Callable:
        """
        动态生成操作方法
        """
        if name.startswith('calc_'):
            operation = name[5:]  # 移除 'calc_' 前缀
            if operation in self.operations:
                return functools.partial(self._execute_operation, self.operations[operation])

        raise AttributeError(f"'{self.__class__.__name__}' object has no attribute '{name}'")

    def _execute_operation(self, operation_func: Callable, *args) -> Any:
        """
        执行操作
        """
        if len(args) >= 2:
            return operation_func(args[0], args[1])
        elif len(args) == 1:
            return operation_func(args[0], 0)
        else:
            return operation_func(0, 0)


class ProxyObject:
    """
    使用 __getattr__ 实现代理模式
    """

    def __init__(self, target: Any):
        self._target = target

    def __getattr__(self, name: str) -> Any:
        """
        代理到目标对象
        """
        print(f"代理访问: {name}")
        attr = getattr(self._target, name)
        if callable(attr):
            def wrapper(*args, **kwargs):
                print(f"调用 {name} 方法前的额外逻辑")
                result = attr(*args, **kwargs)
                print(f"调用 {name} 方法后的额外逻辑")
                return result
            return wrapper
        return attr

    13.7. 使对象可以像函数一样被调用(__call__)¶

    __call__ 使对象能够像函数一样被调用，实现了可调用对象的概念。

    使用场景:

    1. 函数式编程: 创建可调用对象作为函数的替代品
    2. 装饰器: 实现基于类的装饰器
    3. 策略模式: 创建不同行为的可调用对象
    4. 状态保持: 在多次调用之间保持状态

    示例：异步请求链:

    import asyncio
import functools
from typing import Any, Callable


class CallableObject:
    """
    演示 __call__ 魔法方法的使用场景
    使对象可以像函数一样被调用
    """

    def __init__(self, name: str):
        self.name = name
        self.call_count = 0

    def __call__(self, *args, **kwargs) -> str:
        """
        使对象可调用

        Args:
            *args: 位置参数
            **kwargs: 关键字参数

        Returns:
            处理结果
        """
        self.call_count += 1
        print(f"对象 {self.name} 被调用第 {self.call_count} 次")
        print(f"接收到参数: args={args}, kwargs={kwargs}")

        if args:
            return f"{self.name} 处理了参数: {args[0]}"
        return f"{self.name} 被调用"


class AsyncCallableObject:
    """
    演示异步 __call__ 的实现
    """

    def __init__(self, name: str):
        self.name = name
        self.call_count = 0

    async def __call__(self, *args, **kwargs) -> str:
        """
        异步调用实现
        """
        self.call_count += 1
        print(f"异步对象 {self.name} 被调用第 {self.call_count} 次")

        # 模拟异步操作
        await asyncio.sleep(0.1)

        if args:
            return f"异步 {self.name} 处理了参数: {args[0]}"
        return f"异步 {self.name} 被调用"


# 异步上下文管理器配合 __call__ 的示例
class AsyncContextCallable:
    """
    支持异步上下文管理和调用的类
    """

    def __init__(self, name: str):
        self.name = name

    async def __aenter__(self):
        print(f"进入异步上下文: {self.name}")
        return self

    async def __aexit__(self, exc_type, exc_val, exc_tb):
        print(f"退出异步上下文: {self.name}")
        return False

    async def __call__(self, value: Any) -> str:
        """
        异步调用实现
        """
        print(f"异步调用 {self.name} 与值: {value}")
        await asyncio.sleep(0.1)  # 模拟异步操作
        return f"处理完成: {value}"

    13.11 任务依赖链 (__lshift__ / __rshift__)¶

    利用 << (左移) 和 >> (右移) 运算符形象地定义对象间的流向或依赖关系，常用于工作流引擎（如 Airflow）或任务编排。

    示例：定义任务执行顺序:

    class Task:
    def __init__(self, name):
        self.name = name
        self.downstream = []

    def __rshift__(self, other):
        # 实现 task1 >> task2
        # 表示 task1 是 task2 的上游 (task1 执行完 -> task2)
        if isinstance(other, list):
             for o in other:
                 self.downstream.append(o)
        else:
            self.downstream.append(other)
        return other  # 返回 other 允许链式调用 t1 >> t2 >> t3

    def __lshift__(self, other):
        # 实现 task1 << task2
        # 表示 task2 是 task1 的上游 (task2 >> task1)
        other.__rshift__(self)
        return other

    def __repr__(self):
        return f"Task({self.name})"

# 使用
t_extract = Task("Extract")
t_transform = Task("Transform")
t_load = Task("Load")

# 链式定义: Extract -> Transform -> Load
t_extract >> t_transform >> t_load

# 或者使用反向引用
# t_load << t_transform << t_extract

print(f"{t_extract.name} downstream: {t_extract.downstream}")
# Extract downstream: [Task(Transform)]

    13.12 自动嵌套字典 (__missing__)¶

    __missing__ 是 dict 子类的专属方法。利用它可实现 Perl 风格的“自动生成”字典（Autovivification），处理多层嵌套数据时非常方便。

    示例：无限层级字典:

    class AutoDict(dict):
    def __missing__(self, key):
        # 当键不存在时，自动创建一个新的 AutoDict 作为值
        value = self[key] = type(self)()
        return value

# 使用
data = AutoDict()
data['users']['id_1']['name'] = 'Alice'
data['users']['id_1']['role'] = 'Admin'

# 无需手动初始化每一层
import json
print(json.dumps(data, indent=2))
# {
#   "users": {
#     "id_1": {
#       "name": "Alice",
#       "role": "Admin"
#     }
#   }
# }

    13.13 极致内存优化 (__slots__)¶

    虽然 __slots__ 不是方法而是一个类属性（Attribute），但它深刻影响对象底层结构。它告诉 Python 不要为实例创建 __dict__，而是为固定属性分配内存。

    示例：百万级对象内存对比:

    import sys

class RegularPoint:
    def __init__(self, x, y):
        self.x = x
        self.y = y

class SlottedPoint:
    __slots__ = ('x', 'y')  # 限制只能有 x 和 y 属性
    def __init__(self, x, y):
        self.x = x
        self.y = y

p_reg = RegularPoint(1, 2)
p_slot = SlottedPoint(1, 2)

# 估算差异 (字节数可能因 Python 版本/OS 而异)
# Regular 包含 __dict__ 和 __weakref__，开销较大
print(sys.getsizeof(p_reg) + sys.getsizeof(p_reg.__dict__))
# 输出示例: ~152 bytes

# Slotted 仅存储固定属性
print(sys.getsizeof(p_slot))
# 输出示例: ~48 bytes

    13.14 函数组合 (__matmul__)¶

    利用 @ 矩阵乘法运算符实现类似数学中的函数组合 $f(g(x))$。

    示例:

    class Composer:
    def __init__(self, func):
        self.func = func

    def __matmul__(self, other):
        # f @ g  => f(g(x))
        return Composer(lambda x: self.func(other.func(x)))

    def __call__(self, x):
        return self.func(x)

def double(x): return x * 2
def increment(x): return x + 1

# 包装函数
f = Composer(double)
g = Composer(increment)

# 组合: h(x) = f(g(x)) = 2 * (x + 1)
h = f @ g

print(h(10))  # 22

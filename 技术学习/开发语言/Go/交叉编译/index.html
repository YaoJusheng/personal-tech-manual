    交叉编译¶

    1.Go编译简介¶

    Go 语言的编译过程非常简洁高效，对开发者极其友好。其主要特点可以概括为：

    1. 开箱即用：安装好 Go 后，只需一个简单的 go build 命令即可完成编译、链接、生成最终可执行文件的所有步骤，无需复杂的配置（如 Makefile）。
    2. 极快的编译速度：这是 Go 的核心设计目标之一。快速的编译速度极大地提升了开发者的迭代效率和幸福感。其速度优势主要来源于：
    3. 清晰的语法和规整的代码规范，使得解析更容易。
    4. 显式导入包，避免了 C/C++ 中头文件解析和包含的巨额开销。
    5. 依赖关系简单，编译器可以并行地编译包。
    6. 静态编译：默认情况下，Go 会将所有依赖（包括标准库和第三方库）都静态链接到一个独立的可执行文件中。这意味着：
    7. 部署简单：只需将二进制文件复制到目标机器即可运行，无需在目标机器上安装运行时环境或依赖包。
    8. 一致性：避免了因环境不同导致的依赖库版本冲突问题（“在我机器上是好的”问题）。
    9. 跨平台编译：轻松编译出运行于其他操作系统和架构的程序。
    10. 只需设置 GOOS (目标操作系统) 和 GOARCH (目标架构) 环境变量即可。
    11. 例如：在 Linux 上编译 Windows 64 位程序：GOOS=windows GOARCH=amd64 go build -o myapp.exe。
    12. 精简的工具链：go 命令集成了所有功能：编译 (build)、运行 (run)、测试 (test)、格式化 (fmt)、依赖管理 (mod) 等，消除了对第三方构建工具的依赖。

    常用编译命令¶

    • go build：编译包和依赖，但不安装结果。默认输出到当前目录。
    • go install：编译并安装包和依赖。将可执行文件安装到 $GOPATH/bin (或 $GOBIN) 目录。
    • go run：编译并直接运行 Go 程序，不会在磁盘上保留可执行文件，适合快速测试。

    2.4.Markfile构建¶

    1.Example¶

    # * @File: Makefile
# * @Description: Building script for example-edge-gui
# * @Author: Jusheng Yao

# Project Information
APP_NAME ?= example-edge-gui
GIT_COMMIT = $(shell git rev-parse --short HEAD 2>/dev/null || echo "unknown")
VERSION = $(shell git describe --tags --always 2>/dev/null || echo "v0.0.1-alpha-$(GIT_COMMIT)")
BUILD_DATE = $(shell date -u +'%Y-%m-%dT%H:%M:%SZ')
GO_VERSION = $(shell go version | awk '{print $$3}' | tr -d 'go')
# Prefer an explicit override (environment or command-line). If not set, try `go env CGO_ENABLED`.
# If that still returns empty, fall back to 0 to avoid accidental CGO-enabled cross-builds.
CGO_ENABLED ?= $(shell go env CGO_ENABLED)
ifeq ($(strip $(CGO_ENABLED)),)
CGO_ENABLED = 0
endif
# Target platforms
PLATFORMS ?= linux/amd64 linux/arm64 darwin/amd64 darwin/arm64 windows/amd64 windows/arm64

# Optional per-platform cross C compiler overrides (leave empty to use system default)
# Example:
#   CC_LINUX_AMD64=x86_64-linux-gnu-gcc
#   CC_LINUX_ARM64=aarch64-linux-gnu-gcc
#   CC_DARWIN_AMD64=o64-clang
#   CC_DARWIN_ARM64=oa64-clang
#   CC_WINDOWS_AMD64=x86_64-w64-mingw32-gcc
#   CC_WINDOWS_ARM64=aarch64-w64-mingw32-gcc
CC_LINUX_AMD64 ?= gcc
CC_LINUX_ARM64 ?= zig cc -target aarch64-linux-gnu
CC_DARWIN_AMD64 ?= zig cc -target x86_64-macos
CC_DARWIN_ARM64 ?= zig cc -target aarch64-macos
CC_WINDOWS_AMD64 ?= zig cc -target x86_64-windows-gnu
CC_WINDOWS_ARM64 ?= zig cc -target aarch64-windows-gnu
LDFLAGS = -s -w \
    -X 'example-edge-gui/pkg/version.Version=$(VERSION)' \
    -X 'example-edge-gui/pkg/version.Commit=$(GIT_COMMIT)' \
    -X 'example-edge-gui/pkg/version.BuildTime=$(BUILD_DATE)' \
    -X 'example-edge-gui/pkg/version.BuildUser=$(shell git config user.name)' \
    -X 'example-edge-gui/pkg/version.GoVersion=$(GO_VERSION)'

# Colors
RED := \033[0;31m
GREEN := \033[0;32m
YELLOW := \033[0;33m
BLUE := \033[0;34m
NC := \033[0m # No Color

# Output directory
DIST_DIR := dist

# Default target
all: build-all

# Build project
build: clean
    @if [ "$(shell gcc -dumpmachine)" != "x86_64-linux-gnu" ]; then echo "[WARN] gcc target $$(gcc -dumpmachine)"; fi
    @echo "$(GREEN)🔧 Building $(APP_NAME) v-$(VERSION) for current platform ($$GOOS/$$GOARCH, CGO: $(CGO_ENABLED)) ⏳...$(NC)"
    @mkdir -p $(DIST_DIR)/$$GOOS/$$GOARCH
    # Remove accidental Windows 32-bit resource object when building non-Windows target
    @if [ "$$GOOS" != "windows" ] && [ -f versioninfo.syso ]; then \
        if file versioninfo.syso | grep -qi '80386'; then \
            echo "$(YELLOW)[warn] Removing 32-bit versioninfo.syso (Windows resource) for non-windows build$(NC)"; \
            rm -f versioninfo.syso; \
        fi; \
    fi

    @echo "$(YELLOW)📂 Copying config and docs ⚙️ ...$(NC)"
    @cp LICENSE $(DIST_DIR)/$$GOOS/$$GOARCH/ || echo "License not found, skipping copy"
    @cp -r docs/images $(DIST_DIR)/$$GOOS/$$GOARCH/ || echo "Images not found, skipping copy"
    @cp docs/GUIDE.md $(DIST_DIR)/$$GOOS/$$GOARCH/README.md || echo "README not found, skipping copy"
    @if [ "$$GOOS" = "linux" ]; then \
        mkdir $(DIST_DIR)/$$GOOS/$$GOARCH/config; \
        cp config/example-edge-gui.desktop $(DIST_DIR)/$$GOOS/$$GOARCH/config; \
    fi || echo "Config not found, skipping copy"

# Select per-platform CC if provided
    @CC_VALUE=""; \
    case "$$GOOS/$$GOARCH" in \
        linux/amd64)   CC_VALUE="$(CC_LINUX_AMD64)" ;; \
        linux/arm64)   CC_VALUE="$(CC_LINUX_ARM64)" ;; \
        darwin/amd64)  CC_VALUE="$(CC_DARWIN_AMD64)" ;; \
        darwin/arm64)  CC_VALUE="$(CC_DARWIN_ARM64)" ;; \
        windows/amd64) CC_VALUE="$(CC_WINDOWS_AMD64)" ;; \
        windows/arm64) CC_VALUE="$(CC_WINDOWS_ARM64)" ;; \
        *) CC_VALUE="$(CC)" ;; \
    esac; \
    echo "$(BLUE)Using CC=$${CC_VALUE:-default}$(NC)"; \
    LDF="$(LDFLAGS)"; \
    if [ "$$GOOS" = "windows" ]; then \
        LDF="$(LDFLAGS) -H windowsgui"; \
        if [ "$$GOARCH" = "amd64" ]; then \
            GOOS=$$GOOS GOARCH=$$GOARCH APP_NAME="$(APP_NAME)" bash scripts/generate-winrc.sh; \
        fi; \
    fi; \
    if [ -n "$$CC_VALUE" ]; then \
        CC="$$CC_VALUE" GOOS=$$GOOS GOARCH=$$GOARCH CGO_ENABLED=$(CGO_ENABLED) go build $(if $(GO_TAGS),-tags "$(GO_TAGS)",) -ldflags "$$LDF" -o $(DIST_DIR)/$$GOOS/$$GOARCH/$(APP_NAME) .; \
    else \
        GOOS=$$GOOS GOARCH=$$GOARCH CGO_ENABLED=$(CGO_ENABLED) go build $(if $(GO_TAGS),-tags "$(GO_TAGS)",) -ldflags "$$LDF" -o $(DIST_DIR)/$$GOOS/$$GOARCH/$(APP_NAME) .; \
    fi
    @if [ "$$GOOS" = "windows" ]; then \
        rm -f versioninfo.json versioninfo.syso || true; \
        mv $(DIST_DIR)/$$GOOS/$$GOARCH/$(APP_NAME) $(DIST_DIR)/$$GOOS/$$GOARCH/$(APP_NAME).exe; \
    fi

    @echo "\n$(BLUE)📦 Packing artifacts...$(NC)"

    @cd $(DIST_DIR)/$$GOOS/$$GOARCH/ && zip -r $(APP_NAME)_$(VERSION)_$$GOOS-$$GOARCH.zip . && mv $(APP_NAME)_$(VERSION)_$$GOOS-$$GOARCH.zip ../.. && cd - > /dev/null
    @echo "\n$(BLUE)📦 Archiving artifacts...$(NC)"
    @tar --warning=no-file-changed  -czvf $(DIST_DIR)/$(APP_NAME)_$(VERSION)_$$GOOS-$$GOARCH.tar.gz --exclude=\*.{jpg,gif,png,wmv,flv,tar.gz,zip} -C $(DIST_DIR)/$$GOOS/$$GOARCH/ .
    @rm -rf $(DIST_DIR)/$$GOOS/
    @echo "\n$(GREEN)✅ Build completed  Output: $(DIST_DIR)$(NC)"

icons:
    @echo "$(BLUE)🎨 Generating icons from SVG...$(NC)"
    bash scripts/generate-icons.sh

build-all: clean
    @echo "$(GREEN)🔧 Building $(APP_NAME) v-$(VERSION) for all platforms (CGO: $(CGO_ENABLED)) ⏳...$(NC)"
    @mkdir -p $(DIST_DIR)
    @set -e; for platform in $(PLATFORMS); do \
        OS=$${platform%/*}; \
        ARCH=$${platform#*/}; \
        BINARY_NAME=$(APP_NAME)_$$OS_$$ARCH; \
        mkdir -p $(DIST_DIR)/$$platform; \
        echo "$(YELLOW)📂 Copying config and docs ⚙️ ...$(NC)"; \
        cp LICENSE $(DIST_DIR)/$$platform/ || echo "License not found, skipping copy"; \
        cp -r docs/images $(DIST_DIR)/$$platform/ || echo "Images not found, skipping copy"; \
        cp docs/GUIDE.md $(DIST_DIR)/$$platform/README.md || echo "README not found, skipping copy"; \
        LDF="$(LDFLAGS)"; \
        if [ "$$OS" = "windows" ]; then \
            BINARY_NAME="$$BINARY_NAME.exe"; \
            LDF="$(LDFLAGS) -H windowsgui"; \
            if [ "$$ARCH" = "amd64" ]; then \
                GOOS=$$OS GOARCH=$$ARCH APP_NAME="$(APP_NAME)" bash scripts/generate-winrc.sh; \
            fi; \
        elif [ "$$OS" = "linux" ]; then \
            mkdir $(DIST_DIR)/$$platform/config; \
            cp config/example-edge-gui.desktop $(DIST_DIR)/$$platform/config || echo "Config not found, skipping copy"; \
        fi; \
        # Select per-platform CC if provided \
        CC_VALUE=""; \
        case "$$platform" in \
            linux/amd64)   CC_VALUE="$(CC_LINUX_AMD64)" ;; \
            linux/arm64)   CC_VALUE="$(CC_LINUX_ARM64)" ;; \
            darwin/amd64)  CC_VALUE="$(CC_DARWIN_AMD64)" ;; \
            darwin/arm64)  CC_VALUE="$(CC_DARWIN_ARM64)" ;; \
            windows/amd64) CC_VALUE="$(CC_WINDOWS_AMD64)" ;; \
            windows/arm64) CC_VALUE="$(CC_WINDOWS_ARM64)" ;; \
            *) CC_VALUE="$(CC)" ;; \
        esac; \
        echo "$(BLUE)Building for platform ($$platform) [CC=$${CC_VALUE:-default}] ⏳...$(NC)"; \
        if [ -n "$$CC_VALUE" ]; then \
            CC="$$CC_VALUE" GOOS=$$OS GOARCH=$$ARCH CGO_ENABLED=$(CGO_ENABLED) go build $(if $(GO_TAGS),-tags "$(GO_TAGS)",) -ldflags "$$LDF" -o $(DIST_DIR)/$$platform/$$BINARY_NAME .; \
        else \
            GOOS=$$OS GOARCH=$$ARCH CGO_ENABLED=$(CGO_ENABLED) go build $(if $(GO_TAGS),-tags "$(GO_TAGS)",) -ldflags "$$LDF" -o $(DIST_DIR)/$$platform/$$BINARY_NAME .; \
        fi; \
        if [ "$$OS" = "windows" ]; then \
            rm -f versioninfo.json versioninfo.syso || true; \
        fi; \
        echo "\n$(BLUE)📦 Packing artifacts...$(NC)"; \
        tar --warning=no-file-changed -czvf $(DIST_DIR)/$$OS/$(APP_NAME)_$(VERSION)_$$OS_$$ARCH.tar.gz --exclude=\*.{jpg,gif,png,wmv,flv,tar.gz,zip} -C $(DIST_DIR)/$$platform/ . && rm -rf $(DIST_DIR)/$$platform; \
    done
    @echo "\n$(GREEN)✅ Build completed  Output: $(DIST_DIR)$(NC)"

clean:
    @echo "$(RED)⚠️ Cleaning build artifacts...$(NC)\n"
    @rm -rf $(DIST_DIR)

test:
    @echo "$(BLUE)🔍 Running tests...$(NC)\n"
    @go test -race -coverprofile=coverage.txt -covermode=atomic ./...

# Build using go releaser
goreleaser:
    @echo "$(YELLOW)⚙️ Building with GoReleaser...$(NC)"
    @goreleaser build --snapshot --clean

release: build-all
    @echo "$(GREEN)🐢 Release $(VERSION) created$(NC)"

# Cross-platform construction auxiliary commands
build-linux:
    @GOOS=linux GOARCH=$${GOARCH:-amd64} make build --silent

build-macos:

    6.自动化脚本¶

    #!/bin/bash

# 动态版本信息生成脚本 (build_windows.sh)
set -e

# 获取版本信息（可以从环境变量、Git标签或其他来源获取）
VERSION=${VERSION:-"1.0.0.0"}
GIT_COMMIT=$(git rev-parse --short HEAD 2>/dev/null || echo "unknown")
BUILD_DATE=$(date -u +"%Y-%m-%dT%H:%M:%SZ")
PRODUCT_NAME=${PRODUCT_NAME:-"My Go Application"}
COMPANY_NAME=${COMPANY_NAME:-"My Company"}

# 解析版本号为各组成部分
MAJOR=$(echo $VERSION | cut -d. -f1)
MINOR=$(echo $VERSION | cut -d. -f2)
PATCH=$(echo $VERSION | cut -d. -f3)
BUILD=$(echo $VERSION | cut -d. -f4)

# 创建动态版本信息JSON
cat > versioninfo.json << EOF
{
  "FixedFileInfo": {
    "FileVersion": {
      "Major": ${MAJOR:-0},
      "Minor": ${MINOR:-0},
      "Patch": ${PATCH:-0},
      "Build": ${BUILD:-0}
    },
    "ProductVersion": {
      "Major": ${MAJOR:-0},
      "Minor": ${MINOR:-0},
      "Patch": ${PATCH:-0},
      "Build": ${BUILD:-0}
    },
    "FileFlagsMask": "3f",
    "FileFlags": "00",
    "FileOS": "040004",
    "FileType": "01",
    "FileSubType": "00"
  },
  "StringFileInfo": {
    "Comments": "基于Go语言开发的应用程序",
    "CompanyName": "${COMPANY_NAME}",
    "FileDescription": "${PRODUCT_NAME}",
    "FileVersion": "${VERSION}",
    "InternalName": "${PRODUCT_NAME}",
    "LegalCopyright": "版权所有 © $(date +%Y) ${COMPANY_NAME}",
    "LegalTrademarks": "${PRODUCT_NAME} 是 ${COMPANY_NAME} 的商标",
    "OriginalFilename": "${PRODUCT_NAME}.exe",
    "PrivateBuild": "Git Commit: ${GIT_COMMIT}",
    "ProductName": "${PRODUCT_NAME}",
    "ProductVersion": "${VERSION}",
    "SpecialBuild": "构建时间: ${BUILD_DATE}"
  },
  "VarFileInfo": {
    "Translation": {
      "LangID": 0x0409,  # 英语(美国)
      "CharsetID": 0x04B0  # Unicode
    }
  },
  "IconPath": "app.ico"  # 添加图标路径
}
EOF

# 生成资源文件
if command -v goversioninfo >/dev/null 2>&1; then
  echo "生成版本资源文件..."
  goversioninfo -o versioninfo.syso versioninfo.json
else
  echo "错误: 未找到 goversioninfo 工具"
  echo "请安装: go install github.com/josephspurrier/goversioninfo/cmd/goversioninfo@latest"
  exit 1
fi

# 交叉编译Windows应用
echo "编译Windows应用程序..."
GOOS=windows GOARCH=amd64 go build -ldflags="\
  -X main.Version=${VERSION} \
  -X main.BuildDate=${BUILD_DATE} \
  -X main.GitCommit=${GIT_COMMIT} \
  -s -w" \
  -o "${PRODUCT_NAME}.exe"

# 清理临时文件
rm -f versioninfo.json versioninfo.syso

echo "构建完成: ${PRODUCT_NAME}.exe"

    其中，VarFileInfo：

    • 中文，中国：
    "VarFileInfo": {
  "Translation": {
    "LangID": 0x0804,   // 简体中文(中国)
    "CharsetID": 0x04B0 // Unicode (UTF-16LE)
  }
}
    • 英文，美国：
    "VarFileInfo": {
  "Translation": [
    {
      "LangID": 0x0409,   // 英语(美国)
      "CharsetID": 0x04B0 // Unicode (UTF-16LE)
    }
  ]
}

    以下是一些常用的语言和字符集标识符：

    语言/地区 LangID 字符集 CharsetID
    简体中文(中国) 0x0804 Unicode 0x04B0
    英语(美国) 0x0409 Unicode 0x04B0
    繁体中文(台湾) 0x0404 Unicode 0x04B0
    日语(日本) 0x0411 Unicode 0x04B0
    韩语(韩国) 0x0412 Unicode 0x04B0

    3.2.方法二：手动创建资源文件¶

    如果您不想使用第三方工具，可以手动创建资源文件：

    1. 创建 versioninfo.rc 文件¶

    #include <windows.h>

// 定义图标资源（IDR_MAINICON 为资源ID，通常用1）
IDR_MAINICON ICON "app_icon.ico"

// 版本信息（与之前的配置一致，确保属性详情完整）
VS_VERSION_INFO VERSIONINFO
FILEVERSION     1,2,3,0
PRODUCTVERSION  1,2,0,0
FILEFLAGSMASK 0x3fL
FILEFLAGS 0x0L
FILEOS VOS__WINDOWS32
FILETYPE VFT_APP
FILESUBTYPE 0x0L
BEGIN
    BLOCK "StringFileInfo"
    BEGIN
        BLOCK "080404B0"  // 中文（中国）
        BEGIN
            VALUE "Comments", "这是应用程序的详细描述\0"
            VALUE "CompanyName", "您的公司名称\0"
            VALUE "FileDescription", "文件描述\0"
            VALUE "FileVersion", "1.2.3.0\0"
            VALUE "InternalName", "内部名称\0"
            VALUE "LegalCopyright", "版权所有 © 2023 您的公司\0"
            VALUE "LegalTrademarks", "商标信息\0"
            VALUE "OriginalFilename", "您的程序.exe\0"
            VALUE "ProductName", "产品名称\0"
            VALUE "ProductVersion", "1.2.0.0\0"
        END
    END
    BLOCK "VarFileInfo"
    BEGIN
        VALUE "Translation", 0x409, 1200
    END
END

    2. 使用 MinGW 编译资源文件¶

    # 安装 MinGW (在 Ubuntu/Debian 上)
sudo apt-get install mingw-w64 mingw-w64-tools

# 编译资源文件
x86_64-w64-mingw32-windres versioninfo.rc -O coff -o versioninfo.syso

# or on windows
cargo add windres

windres -i versioninfo.rc -O coff -o versioninfo.syso

    自动化构建脚本示例¶

    创建一个 build.sh 脚本来自动化整个过程：

    #!/bin/bash

# 生成版本资源文件
goversioninfo -o versioninfo.syso versioninfo.json

# 交叉编译
GOOS=windows GOARCH=amd64 go build -ldflags="-s -w" -o your_app.exe

# 清理
rm -f versioninfo.syso

echo "构建完成: your_app.exe"

    3.3.验证版本信息¶

    编译完成后，您可以在Windows上右击生成的exe文件，选择"属性"->"详细信息"查看嵌入的版本信息。

    3.4.注意事项¶

    1. 图标文件：如果需要添加应用程序图标，可以在versioninfo.JSON中指定IconPath字段
    2. 字符编码：确保使用正确的字符编码，推荐使用UTF-8
    3. 版本号格式：Windows版本号通常采用四段式（主版本.次版本.修订版本.构建号）
    4. 清理：建议在构建完成后删除.syso文件，以免影响非Windows平台的编译

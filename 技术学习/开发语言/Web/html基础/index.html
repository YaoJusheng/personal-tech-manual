    核心概念与语言特性¶

    • HTML（HyperText Markup Language） 是 Web 文档的结构化描述语言，负责定义页面骨架与语义。
    • HTML5 引入语义化标签、多媒体支持、离线存储、表单增强等现代特性。
    • 现代 HTML 需配合 CSS/JS、ARIA、SEO 元数据等共同构建可访问、高性能的应用。
    • 文档对象模型（DOM）：HTML 的树形结构表示，每个元素都是 DOM 树的一个节点。

    HTML 文档结构详解¶

    基础文档模板¶

    <!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="utf-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1" />
    <meta name="description" content="页面描述，用于SEO" />
    <meta name="keywords" content="关键词1,关键词2" />
    <meta name="author" content="作者名称" />
    <title>页面标题 - 网站名称</title>

    <!-- CSS 样式表 -->
    <link rel="stylesheet" href="/assets/css/main.css" />

    <!-- Favicon 图标 -->
    <link rel="icon" type="image/x-icon" href="/favicon.ico" />

    <!-- Open Graph 标签，用于社交分享 -->
    <meta property="og:title" content="页面标题" />
    <meta property="og:description" content="页面描述" />
    <meta property="og:image" content="/assets/images/og-image.jpg" />
    <meta property="og:url" content="https://example.com/page" />
</head>
<body>
    <!-- 页面头部 -->
    <header role="banner">
        <nav role="navigation" aria-label="主导航">
            <ul>
                <li><a href="/" aria-current="page">首页</a></li>
                <li><a href="/about">关于我们</a></li>
                <li><a href="/contact">联系我们</a></li>
            </ul>
        </nav>
    </header>

    <!-- 主要内容区域 -->
    <main id="main-content" role="main">
        <article>
            <header>
                <h1>文章标题</h1>
                <p class="meta">
                    <time datetime="2024-01-30">2024年1月30日</time>
                    <span>作者：<a href="/authors/john-doe">John Doe</a></span>
                </p>
            </header>

            <section>
                <h2>章节标题</h2>
                <p>这里是正文内容...</p>
            </section>
        </article>
    </main>

    <!-- 侧边栏 -->
    <aside role="complementary">
        <section>
            <h3>相关链接</h3>
            <ul>
                <li><a href="/related-article-1">相关文章1</a></li>
                <li><a href="/related-article-2">相关文章2</a></li>
            </ul>
        </section>
    </aside>

    <!-- 页面底部 -->
    <footer role="contentinfo">
        <p>&copy; 2024 网站名称. 保留所有权利.</p>
    </footer>

    <!-- JavaScript 脚本 -->
    <script type="module" src="/assets/js/main.js"></script>

    <!-- 无脚本提示 -->
    <noscript>
        <p>本页面需要启用 JavaScript 才能正常运行。</p>
    </noscript>
</body>
</html>

    文档类型声明（DOCTYPE）¶

    <!-- HTML5 标准模式 -->
<!DOCTYPE html>

<!-- HTML 4.01 严格模式 -->
<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN" "http://www.w3.org/TR/html4/strict.dtd">

<!-- XHTML 1.0 严格模式 -->
<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">

    使用场景分析：

    • HTML5 模式：现代 Web 应用的标准，提供最佳兼容性和性能
    • HTML4/XHTML：遗留系统维护，需要特定 DTD 验证

    <!-- 布尔属性 -->
<input type="text" required disabled />

<button type="submit" formmethod="post">提交</button>

<!-- 数据属性 -->
<div data-product-id="12345" data-category="electronics">
    产品信息
</div>

<nav aria-label="主导航">
    <ul role="menubar">
        <li role="none"><a role="menuitem" href="/">首页</a></li>
    </ul>
</nav>

    表单最佳实践总结¶

    可访问性优化：

    • 使用 label 标签关联所有表单控件
    • 添加 aria-describedby 提供额外帮助信息
    • 设置 aria-invalid 指示验证状态
    • 使用 role="alert" 显示错误信息
    • 确保键盘导航和屏幕阅读器兼容

    用户体验优化：

    • 选择合适的 input type 启用浏览器原生验证
    • 设置 autocomplete 属性提升填写效率
    • 提供实时验证反馈
    • 使用 placeholder 提供输入提示
    • 设置合理的 minlength/maxlength 限制

    安全性考虑：

    • 添加 CSRF 令牌防止跨站请求伪造
    • 验证所有服务器端输入
    • 使用 pattern 属性进行客户端验证
    • 避免在客户端存储敏感信息
    输入类型 用途 验证特性 适用场景
    email 邮箱地址 格式验证 用户注册、联系表单
    tel 电话号码 数字键盘 客服联系、订单确认
    url 网址 URL格式验证 个人网站、社交链接
    number 数值 范围限制 年龄、数量选择
    date 日期 日期选择器 出生日期、预约时间
    search 搜索 搜索历史 站内搜索、过滤器

<script>
// localStorage 增强
class EnhancedStorage {
  static set(key, value, expiry = null) {
    const data = {
      value: value,
      expiry: expiry,
      timestamp: Date.now()
    };
    localStorage.setItem(key, JSON.stringify(data));
  }

  static get(key) {
    try {
      const item = localStorage.getItem(key);
      if (!item) return null;

      const data = JSON.parse(item);
      if (data.expiry && Date.now() > data.expiry) {
        localStorage.removeItem(key);
        return null;
      }
      return data.value;
    } catch (error) {
      console.error('Storage error:', error);
      return null;
    }
  }
}

// Intersection Observer 实现懒加载
const observer = new IntersectionObserver((entries, observer) => {
  entries.forEach(entry => {
    if (entry.isIntersecting) {
      const element = entry.target;

      // 处理图片
      if (element.tagName === 'IMG') {
        element.src = element.dataset.src;
        element.classList.remove('lazy-image');
      }

      // 处理视频
      if (element.tagName === 'VIDEO') {
        const source = element.querySelector('source');
        if (source) {
          source.src = source.dataset.src;
          element.load();
        }
      }

      // 处理 iframe
      if (element.tagName === 'IFRAME') {
        element.src = element.dataset.src;
      }

      observer.unobserve(element);
    }
  });
}, {
  rootMargin: '50px 0px',
  threshold: 0.1
});

// 观察所有懒加载元素
document.querySelectorAll('[data-src]').forEach(element => {
  observer.observe(element);
});
</script>

// 模板渲染函数
class TemplateRenderer {
  constructor(templateId) {
    this.template = document.getElementById(templateId);
  }

  render(data, container) {
    const fragment = this.template.content.cloneNode(true);

    // 数据绑定
    Object.keys(data).forEach(key => {
      const element = fragment.querySelector(`[data-bind="${key}"]`);
      if (element) {
        element.textContent = data[key];
      }
    });

    container.appendChild(fragment);
  }

  renderList(dataList, container, templateId = null) {
    const template = templateId ?
      document.getElementById(templateId) :
      this.template;

    dataList.forEach(data => {
      this.render(data, container);
    });
  }
}

<script>
// ARIA 状态管理
class AriaManager {
  static setInvalid(element, message) {
    element.setAttribute('aria-invalid', 'true');
    const errorElement = document.getElementById(element.id + '-error');
    if (errorElement) {
      errorElement.textContent = message;
      errorElement.style.display = 'block';
    }
  }

  static setValid(element) {
    element.setAttribute('aria-invalid', 'false');
    const errorElement = document.getElementById(element.id + '-error');
    if (errorElement) {
      errorElement.textContent = '';
      errorElement.style.display = 'none';
    }
  }

  static announce(message) {
    const statusElement = document.querySelector('[role="status"]');
    if (statusElement) {
      statusElement.textContent = message;
    }
  }

  static setProgressbar(element, value, max = 100) {
    element.setAttribute('aria-valuenow', value);
    element.setAttribute('aria-valuemax', max);
    element.textContent = `${value}%`;
  }
}

// 使用示例
const emailInput = document.getElementById('email');
emailInput.addEventListener('blur', function() {
  if (!this.value.includes('@')) {
    AriaManager.setInvalid(this, '请输入有效的邮箱地址');
  } else {
    AriaManager.setValid(this);
  }
});
</script>

<script>
// 焦点管理器
class FocusManager {
  constructor(container) {
    this.container = container;
    this.focusableElements = container.querySelectorAll(
      'button, [href], input, select, textarea, [tabindex]:not([tabindex="-1"])'
    );
    this.firstElement = this.focusableElements[0];
    this.lastElement = this.focusableElements[this.focusableElements.length - 1];
  }

  trapFocus(event) {
    if (event.key !== 'Tab') return;

    if (event.shiftKey) {
      // Shift + Tab (向前)
      if (document.activeElement === this.firstElement) {
        event.preventDefault();
        this.lastElement.focus();
      }
    } else {
      // Tab (向后)
      if (document.activeElement === this.lastElement) {
        event.preventDefault();
        this.firstElement.focus();
      }
    }
  }

  setInitialFocus() {
    this.firstElement.focus();
  }

  releaseFocus() {
    // 恢复焦点到触发元素
    const triggerElement = document.querySelector('[data-modal-trigger]');
    if (triggerElement) {
      triggerElement.focus();
    }
  }
}

// 模态对话框管理器
class ModalManager {
  constructor(modalElement) {
    this.modal = modalElement;
    this.focusManager = new FocusManager(modalElement);
    this.previousActiveElement = null;

    this.bindEvents();
  }

  bindEvents() {
    // ESC 键关闭
    document.addEventListener('keydown', (event) => {
      if (event.key === 'Escape' && this.isOpen()) {
        this.close();
      }
    });

    // 点击遮罩层关闭
    document.addEventListener('click', (event) => {
      if (event.target === this.modal) {
        this.close();
      }
    });
  }

  open() {
    this.previousActiveElement = document.activeElement;
    this.modal.style.display = 'flex';
    this.modal.setAttribute('aria-hidden', 'false');
    this.focusManager.setInitialFocus();

    // 焦点陷阱
    this.modal.addEventListener('keydown', (event) => {
      if (event.key === 'Tab') {
        this.focusManager.trapFocus(event);
      }
    });
  }

  close() {
    this.modal.style.display = 'none';
    this.modal.setAttribute('aria-hidden', 'true');
    this.focusManager.releaseFocus();
  }

  isOpen() {
    return this.modal.style.display !== 'none';
  }
}

// 使用示例
const modal = document.querySelector('.modal');
const modalManager = new ModalManager(modal);

// 打开模态框
document.querySelector('[data-modal-trigger]').addEventListener('click', () => {
  modalManager.open();
});
</script>

// 实时文本更新管理器
class LiveRegionManager {
  constructor(container) {
    this.container = container;
    this.setupLiveRegion();
  }

  setupLiveRegion() {
    // 创建隐藏的实时区域
    const liveRegion = document.createElement('div');
    liveRegion.setAttribute('aria-live', 'polite');
    liveRegion.setAttribute('aria-atomic', 'true');
    liveRegion.className = 'sr-only';
    liveRegion.style.cssText = `
      position: absolute;
      width: 1px;
      height: 1px;
      padding: 0;
      margin: -1px;
      overflow: hidden;
      clip: rect(0, 0, 0, 0);
      white-space: nowrap;
      border: 0;
    `;
    document.body.appendChild(liveRegion);
    this.liveRegion = liveRegion;
  }

  announce(message) {
    this.liveRegion.textContent = message;
  }

  addMessage(sender, content, timestamp) {
    const messageDiv = document.createElement('div');
    messageDiv.className = 'message';
    messageDiv.innerHTML = `
      <span class="sender">${sender}:</span>
      <span class="content">${content}</span>
      <time class="timestamp" datetime="${timestamp}">
        ${new Date(timestamp).toLocaleTimeString()}
      </time>
    `;

    this.container.appendChild(messageDiv);

    // 通知屏幕阅读器
    this.announce(`${sender} 说：${content}`);
  }
}

    SEO 与元信息¶

    标签/属性 作用
    <meta name="description"> 设置摘要，提升搜索结果点击率
    <link rel="canonical"> 指定权威 URL，防止重复内容
    <meta property="og:*"> Open Graph 协议，优化分享卡片
    <meta name="theme-color"> 移动端地址栏配色
    structured data (JSON-LD) 结构化数据（FAQ/Breadcrumb/Product）

    资源加载策略¶

    • defer / async：延迟或异步加载脚本，避免阻塞 HTML 解析。
    • preload / prefetch：预加载关键资源，或预取下一页面所需资源。
    • lazy-loading：loading="lazy" 延迟图片/iframe。
    • Content-Security-Policy：限制资源来源，提升安全性。

    工具链与调试¶

    • 浏览器 DevTools：Elements 面板查看 DOM/Accessibility 树，Network 面板检查资源顺序。
    • Lint/格式化：使用 HTMLHint、Prettier 统一书写规范。
    • 静态站点生成（SSG）：如 Astro、Eleventy，输出语义化 HTML。
    • 自动化测试：结合 Playwright/Puppeteer 进行 DOM 快照与可访问性扫描（axe-core）。

    常见检查清单¶

    1. 是否设置 lang、charset、viewport 等基础元信息？
    2. 结构是否清晰、语义化，是否滥用 <div>？
    3. 表单与交互是否提供 label、键盘操作与错误提示？
    4. 图片/媒体是否具备替代文本与懒加载策略？
    5. 是否启用了安全响应头（CSP、Referrer-Policy、Permissions-Policy）？

    概述¶

    React是由Facebook开发并维护的一个用于构建用户界面的JavaScript库。它采用组件化架构，支持声明式编程，让开发者能够更轻松地创建交互式用户界面。

    核心特性¶

    1. 组件化架构¶

    • 可复用组件：将UI拆分为独立的、可复用的组件
    • 组件组合：通过组件嵌套和组合构建复杂的用户界面
    • 组件隔离：每个组件拥有自己的状态和逻辑

    2. 声明式编程¶

    • 描述UI外观：开发者描述组件在不同状态下的外观
    • React自动渲染：React根据状态变化自动更新DOM
    • 减少样板代码：无需手动操作DOM更新

    3. 虚拟DOM¶

    • 内存中的DOM表示：React在内存中维护虚拟DOM树
    • 高效更新：只更新实际变化的DOM部分
    • 性能优化：减少不必要的DOM操作

    4. 单向数据流¶

    • 数据向下传递：父组件通过props向子组件传递数据
    • 状态管理：组件内部使用state管理本地状态
    • 可预测性：数据流动方向清晰，易于调试

    React生态系统¶

    核心库¶

    • React：核心库，提供组件和虚拟DOM功能
    • React DOM：React与DOM交互的渲染器
    • React Native：用于移动应用开发的React变体

    开发工具¶

    • Create React App：官方脚手架工具
    • React Developer Tools：浏览器扩展调试工具
    • React Fiddle：在线代码编辑和测试环境

    状态管理¶

    • Redux：可预测的状态容器
    • MobX：响应式状态管理
    • Context API：React内置的上下文状态管理

    路由管理¶

    • React Router：官方路由解决方案
    • Reach Router：无障碍路由库

    UI组件库¶

    • Material-UI：Google Material Design组件
    • Ant Design：企业级UI设计语言
    • Chakra UI：简单、模块化的UI组件

// State使用
class Counter extends React.Component {
  constructor(props) {
    super(props);
    this.state = {
      count: 0
    };
  }

  increment = () => {
    this.setState(prevState => ({
      count: prevState.count + 1
    }));
  };

  decrement = () => {
    this.setState(prevState => ({
      count: prevState.count - 1
    }));
  };

  render() {
    return (
      <div>
        <h2>Count: {this.state.count}</h2>
        <button onClick={this.increment}>+</button>
        <button onClick={this.decrement}>-</button>
      </div>
    );
  }
}

// 函数组件中的State (Hooks)
import { useState } from 'react';

function Counter() {
  const [count, setCount] = useState(0);

  return (
    <div>
      <h2>Count: {count}</h2>
      <button onClick={() => setCount(count + 1)}>+</button>
      <button onClick={() => setCount(count - 1)}>-</button>
    </div>
  );
}

function todoReducer(state, action) {
  switch (action.type) {
    case 'ADD_TODO':
      return {
        ...state,
        todos: [...state.todos, {
          id: Date.now(),
          text: action.text,
          completed: false
        }]
      };
    case 'TOGGLE_TODO':
      return {
        ...state,
        todos: state.todos.map(todo =>
          todo.id === action.id
            ? { ...todo, completed: !todo.completed }
            : todo
        )
      };
    case 'DELETE_TODO':
      return {
        ...state,
        todos: state.todos.filter(todo => todo.id !== action.id)
      };
    case 'SET_FILTER':
      return {
        ...state,
        filter: action.filter
      };
    default:
      return state;
  }
}

function useLocalStorage(key, initialValue) {
  const [storedValue, setStoredValue] = useState(() => {
    try {
      const item = window.localStorage.getItem(key);
      return item ? JSON.parse(item) : initialValue;

      return initialValue;

      window.localStorage.setItem(key, JSON.stringify(value));

  };

}

      logErrorToService(error, errorInfo);

      return (
        <div style={{ padding: '20px', backgroundColor: '#ffe6e6' }}>
